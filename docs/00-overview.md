# 00-overview.md — 프로젝트 전체 개요

## 프로젝트 한 줄 정의

> **SStock** — 한국투자증권(KIS) 국내주식 OpenAPI를 사용해 **실시간 시세 모니터링**과 **룰 기반 자동 매매**를 하나의 상시 구동 서버 + 웹 대시보드로 제공하는 자동 매매 봇.

---

## 핵심 결정 (확정)

| 항목 | 결정 | 비고 |
|------|------|------|
| 백엔드 | Python (FastAPI + asyncio) | 봇 엔진을 asyncio 백그라운드 태스크로 구동 |
| 프론트엔드 | React + TypeScript (Vite) | 실시간 대시보드 |
| 데이터 소스 | 한국투자증권(KIS) 국내주식 OpenAPI | `Reference/API_DOC/` 명세 기준 |
| 거래 모드 | **모의투자 먼저** | 실전 전환은 환경변수로만, 명시적 확인 절차 |
| 매매 전략 | 룰 기반 (설정 가능) | 이동평균·RSI 등 지표 규칙을 설정값으로 구성 |
| 실행 형태 | 상시 구동 서버 + 웹 대시보드 | 로컬 PC 우선, 봇 ON/OFF 수동 토글 |
| 저장소 | SQLite | 체결·신호·포지션·로그 |

## 가장 중요한 원칙 — 안전 우선

이 프로젝트는 **실제 자금이 집행될 수 있는 매매 시스템**이다.

1. 기본 동작은 **모의투자**다. 실전은 환경변수 + 명시적 확인이 동시에 충족될 때만 활성화된다.
2. KIS `appkey`/`appsecret`/`access_token`은 **절대 코드·로그·git에 노출하지 않는다** (환경변수만).
3. 봇은 **수동 시작/정지** 토글과 **일일 주문 한도** 가드를 가진다.
4. 모든 주문 시도·체결·전략 신호는 변경 불가 로그로 기록한다.
