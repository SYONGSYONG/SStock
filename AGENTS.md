# AGENTS.md

## 작업 전 문서 읽기 규칙

- 앞으로 모든 작업은 시작 전에 반드시 `CLAUDE.md`를 먼저 읽는다.
- 앞으로 모든 작업은 `docs` 문서를 아래 순서대로 읽은 뒤 진행한다.
- 기본 읽기 순서:
  1. `CLAUDE.md`
  2. `docs/00-overview.md`
  3. `docs/01-product.md`
  4. `docs/02-specs.md`
  5. `docs/03-design.md`
  6. `docs/04-conventions.md`
  7. `docs/05-tasks.md`
- 작업이 추천 기능과 직접 관련되면 `docs/06-recommend.md`를 추가로 읽는다.
- 작업이 차트 기능과 직접 관련되면 `docs/07-chart.md`를 추가로 읽는다.
- 문서와 구현이 충돌하면 먼저 문서를 기준으로 차이를 식별하고, 그 사실을 사용자에게 명확히 알린다.
